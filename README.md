## О проекте

**404 dev** — это не просто ещё один форум.  
Мы строим **дружелюбное, структурированное и доступное сообщество** для разработчиков, которые устали от токсичности, спама и устаревших платформ.

Проект создаётся с одной простой мыслью:  
> *«Здесь не страшно задать вопрос»*

Никакого осуждения за «глупые» ошибки. Только живые люди, готовые помочь, и удобная база знаний, которая растёт вместе с вами.

### 🎯 Кто наша аудитория

- Молодые разработчики (18–35 лет)
- Выпускники курсов и самоучки
- Начинающие специалисты, ищущие ментора
- Все, кто хочет пройти собеседование, разобраться в сложной теме или получить ревью кода

## Цели проекта

1. **Свободное пространство**  
   — без географических и технологических барьеров

2. **Структурированная база знаний**  
   — объединяем непринуждённость мессенджера с мощью форума

3. **Ответы легко найти**  
   — понятная навигация и поиск

4. **Вопросы не страшно задавать**  
   — дружелюбная атмосфера без осуждения
